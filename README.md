# MobilStock

Application de gestion de boutique dédiée au stock, aux ventes, aux utilisateurs et au suivi d'activité.

## Fonctionnalités principales

- Gestion du stock
- Enregistrement des ventes
- Historique et statistiques
- Gestion des utilisateurs et des rôles
- Paramètres de boutique

## Stack technique

- React
- TypeScript
- Vite
- Supabase
- TanStack Query
- Zustand

## Lancement du projet

```bash
npm install
npm run dev
```

## Vérifications

```bash
npm run lint
npm run build
```
